This file lists people who have opted in to being credited for their
contributions to this unofficial, fan-made companion site.

`support@mecchachameleon.art`. Anonymous contributors are welcome and
do **not** need to be listed here.

### How to opt in

Open a PR that edits this file and adds your line under "Contributors":

```md
- @your-github-handle — what you contributed (e.g. "added 3 hiding spots
  on the Vintage Room map" or "added the zh-CN translation")
```

We will merge it. If you want to stay anonymous, simply don't edit this
file — your contribution still counts.
