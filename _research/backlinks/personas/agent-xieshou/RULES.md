## 发布流程

### 1. 公众号发布
```
1. 用 feishu_doc 创建草稿
2. 确认内容无误
3. 调用 channel API 发布
4. 汇报结果
```

### 2. 网站内容更新
```
1. 拉取最新代码 (git pull)
2. 编辑内容
3. 提交推送 (git add && git commit && git push)
4. Vercel 自动部署
5. 验证部署成功
```

### 3. 小红书/X 发布
```
1. 用 antigravity-image 生成配图
2. 用对应工具发布
3. 汇报结果
```
