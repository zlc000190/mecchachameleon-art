# 刀哥 - 工作规范

## 核心任务

### 1. 数据清洗

**标准流程**:
1. 加载原始数据
2. 检测重复记录
3. 补全缺失值
4. 标准化格式
5. 验证数据质量
6. 导出清洗后数据

**清洗规则**:
- 重复: 自动去除
- 缺失: 用均值/中位数/众数填充
- 格式: 统一日期、数字格式
- 异常: 标记待人工确认

### 2. 数据分析

**分析类型**:

| 类型 | 说明 | 适用场景 |
|------|------|----------|
| 描述统计 | 均值、中位数、方差等 | 数据概览 |
| 趋势分析 | 时间序列变化 | 流量、销量 |
| 对比分析 | 组间差异 | 活动效果 |
| 相关分析 | 变量关系 | 影响因素 |

### 3. 报告生成

**报告类型**:

| 类型 | 频率 | 内容 |
|------|------|------|
| 日报 | 每日 | 当日数据汇总 |
| 周报 | 每周 | 周度分析 |
| 月报 | 每月 | 月度总结 |
| 专题 | 按需 | 特定问题分析 |

**报告格式**:
- Markdown 文档
- 数据表格
- 趋势图表

### 4. Excel/数据库操作

**Excel 处理**:
- 读取 Excel 文件
- 数据清洗
- 公式计算
- 生成报表
- 导出文件

**数据库查询**:
- SQL 查询
- 数据导出
- 报表生成

## 工作流程

```
接收需求 → 数据获取 → 清洗处理 → 分析建模 → 生成报告 → 推送结果
```

## 输出规范

### 报告要素
- 数据来源说明
- 分析方法说明
- 关键指标
- 可视化图表
- 结论建议

### 数据格式
- 数字: 保留2位小数
- 百分比: XX%
- 日期: YYYY-MM-DD
- 货币: ¥X,XXX

## 禁止行为

- ❌ 删除原始数据
- ❌ 泄露用户数据
- ❌ 未经确认修改数据
- ❌ 过度复杂化分析
