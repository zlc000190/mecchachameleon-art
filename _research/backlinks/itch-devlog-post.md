I built mecchachameleon.art — a fan-made source-code learning portal for indie games.

What it does:

- 44 free HTML5 indie games playable in-browser (curated, all from itch.io developers who've opted into HTML5 embeds)
- 4 Godot starter projects with source code linked back to the original creators (Learn Godot's GDScript From Zero, Godot 4 Beginner Pack, Godot Coloring Game Template, Wordle English)
- Every page credits the original developer and links to their itch.io/Steam page
- Multilingual (en / ru)

The pitch: source-code learning is underserved vs. pure "play free games" sites. Most game collections don't credit the developer or link to source. mecchachameleon.art does both.

Why I'm posting this on itch.io devlog specifically:
- This community already builds open-source-friendly games. If anyone wants to contribute a "build this game in 1 hour" tutorial for one of the Godot projects I'm linking to, drop me a line — I'll add it to the page.
- The 4 source-code games I currently link to all have permissive licenses. If you're the original developer of one of them and you'd rather I not link, also drop me a line and I'll remove immediately.
- If you have a Godot/Ren'Py/GB Studio/PICO-8/Bevy project with HTML5 export + source available, consider submitting it to the itch.io "free + platform-web" tag — that's how I found most of the games in the pool.

Tech: Next.js (shipany template), deployed on Dokploy. ~50 pages so far, multilingual content pipeline.

Unofficial / fan-made: mecchachameleon.art is not affiliated with the developers of the games it links to. All credit goes to the original creators.
