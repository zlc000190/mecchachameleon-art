Title: Show HN: mecchachameleon.art – free HTML5 indie games + Godot source code

URL: https://mecchachameleon.art

Body:

Hi HN,

I built mecchachameleon.art as a fan-made source-code learning portal for indie games. The idea: take games that already ship with source (Godot, Ren'Py, GB Studio, PICO-8, Bevy) and re-host the source as a learn-by-cloning resource, with credit back to the original devs.

What's there now:
- 44 free HTML5 indie games playable in-browser, hand-picked from itch.io
- 4 Godot starter projects with source code linked back to the original creators (Learn Godot's GDScript From Zero / Godot 4 Beginner Pack / Godot Coloring Game Template / Wordle English)
- Fan-made "unofficial" framing — every page credits the original developer and links to their itch.io/Steam page
- Multilingual (en / ru / more planned)

Stack: Next.js (shipany template), deploys to Dokploy. ~50 pages with Source Code / How to Play / Credits sections per game.

Why this might be interesting:
- Source-code learning is underserved vs. pure "play free games" sites. Most game collections don't credit the original developer or link to the source.
- The fan-made framing keeps this clearly non-commercial — I don't host the games, I iframe itch.io's HTML5 embed.

What I haven't done: hand-curated "build this game in 1 hour" tutorials per source code. That's the next step — if anyone wants to contribute a tutorial for one of the Godot games I'm linking to, drop me a line.

Not looking for traffic, mostly wanted to share the concept. Happy to answer technical questions about the stack or the curation pipeline.
